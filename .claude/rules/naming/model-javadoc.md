# 普通模型类 JavaDoc 规范

适用于：无 `@Table` 注解、类名不以 `DTO`/`VO` 结尾的普通模型类（如 `AsrResult`、`SetVolumeMessage`）。

## 字段注释规则

摘要行为一个简短句子，解释字段含义，不含标点符号。

需要补充说明时使用 `<h3>字段说明`，格式要求：
- 摘要行与标签之间保留一个空行
- 说明内容用 `<p>` 标签开头，每段独立一行，段内不换行

```java
// ✅ 正确：仅摘要行
/** 操作类型 */
private String action;

// ✅ 正确：摘要行 + 字段说明
/**
 * 操作类型
 *
 * <h3>字段说明
 * <p>音量控制操作类型，支持 set（设置到指定音量）、increase（增加音量）、decrease（减少音量）
 */
private String action;

// ✅ 正确：多段说明
/**
 * 音量级别
 *
 * <h3>字段说明
 * <p>设备需要设置的音量值，范围为 0-100。
 * <p>超出范围时由设备端自行裁剪到合法区间。
 */
private Integer volume;

// ❌ 错误：摘要行含标点
/** 操作类型。 */

// ❌ 错误：摘要行与标签之间缺少空行
/**
 * 操作类型
 * <h3>字段说明
 * <p>音量控制操作类型
 */
```
