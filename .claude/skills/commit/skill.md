# Git Commit Skill

## Description

自动化 Git 代码提交流程，将当前工作区的所有修改内容合并为单次提交，并推送到远程仓库。遵循项目 Git 提交规范，确保提交消息清晰、准确且符合中文最佳实践。

## Use Cases

- 完成功能开发后提交代码
- 修复 bug 后提交代码
- 重构代码后提交代码
- 任何需要将代码变更持久化到 Git 仓库的场景

## Usage

当用户表达以下意图时，自动触发此 skill：
- "提交代码"
- "commit 代码"
- "提交修改"
- "推送代码"
- "git commit"
- 其他表达需要提交代码到 git 仓库的类似表述
